### Estructura del Proyecto

- **`_config.yml`:** Archivo de configuración de Jekyll.
- **`_config_dev.yml`:** Configuración de desarrollo.
- **`_includes/`:** Componentes HTML reutilizables.
- **`_layouts/`:** Plantilla HTML.
- **`_programas/`:** Archivos de programas.
- **`_posts/`:** Archivos de blog.
- **`assets/`:** Recursos como imágenes, CSS y JavaScript.
- **`index.md`:** Página principal del sitio.
- **`scripts/`:** Scripts ejecutables.

## Características Principales

### Desarrollo y Personalización

- **Framework Jekyll:** Utilizamos Jekyll para generar el sitio web estático, lo que facilita la creación y gestión de contenido.
- **HTML y SCSS:** Estructura y diseño del sitio con HTML y SCSS para una personalización completa y estilos modernos.
- **JavaScript:** Añadimos funcionalidades interactivas y dinámicas utilizando JavaScript.

### Funcionalidades de JavaScript

- **Botón de Copiar en Bloques de Código:** Permite a los usuarios copiar fácilmente el código mostrado en los ejemplos.
- **Navegación Interna en Nuevas Páginas:** Facilita la navegación dentro del sitio sin recargar la página.
- **Acordeones:** Componentes interactivos que muestran u ocultan contenido.
- **Generador de Imagen de Cover de Playlist:** Herramienta para crear imágenes personalizadas para listas de reproducción.
- **Consola JavaScript Interactiva:** Consola embebida que permite a los usuarios ejecutar y probar código JavaScript en tiempo real.
- **Recopilación de Datos del Agente de Usuario:** Recoge información del navegador y dispositivo del usuario para análisis.

### Blog y Diario

- **Blog Integrado:** Plataforma de blogging para publicar artículos, noticias y entradas de diario.
- **Sistema de Bitácora:** Funcionalidad para llevar un registro diario o de proyectos, con opciones de categorización y archivo.

### Compatibilidad y Personalización

- **Compatibilidad Multiplataforma:** El sitio es compatible con todos los navegadores modernos y se adapta a diferentes dispositivos y tamaños de pantalla.
- **Fácil Personalización:** Estructura de archivos y configuración intuitiva para que puedas adaptar el sitio a tus necesidades específicas.
- **Desplegable en cualquier Servidor:** Puede ser desplegado en GitHub Pages, Netlify, Vercel, y cualquier servidor compatible con HTML estático.
